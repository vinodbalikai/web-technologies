Java:
Java is a High level Object Oriented Programming language.
HTML:
HTML is a standard markup language used to create and structure of a web page.
CSS:
CSS is used to style and design web pages, including layout, colors, and fonts.
JavaScript:
JavaScript is a scripting language used to create dynamic and interactive web content.
Servlet:
Servlet is a Java-based technology used to create web applications and handle server-side requests.
JDBC:
JDBC is a Java API used to connect and interact with databases.
Spring:
Spring is a powerful Java framework used for building enterprise-level applications.
React:
React is a JavaScript library used to build user interfaces, especially single-page applications.
